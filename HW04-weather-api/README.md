# HW04 

## Description

Create the application that print weather from [https://www.metaweather.com/api/](https://www.metaweather.com/api/) by 
city location.

## Build

To build the program run command:

```
cmake
make weather
```

## Usage

To use the program run command:

```
./weather [CITY]
```

`[CITY]` - input city
